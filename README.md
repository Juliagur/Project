# Project
# Интеллектуальный ассистент с вызовом функций  

Этот проект демонстрирует создание **локального ИИ-ассистента**, способного:
- Понимать естественные запросы пользователя,
- **Вызывать инструменты (Tools)** для выполнения действий (проверка погоды, бронирование и др.),
- **Искать информацию в базе знаний (Retrieval)**,
- **Запоминать историю диалога (Memory)**,
- Работать **без интернета** после первого запуска.

Архитектура построена на **LangChain** и использует **Phi-3-mini** — компактную, но мощную модель от Microsoft.

---

## ✨ Возможности

- **Function Calling**: автоматическое определение нужной функции и извлечение параметров из текста.
- **Custom Tool**: пример пользовательского инструмента (`get_current_time`).
- **Retrieval**: ответы на часто задаваемые вопросы из встроенной базы знаний.
- **Memory**: контекстное понимание на основе истории диалога.
- **Chain**: кастомная цепочка обработки запроса на основе `langchain_core.chains.Chain`.

---

## ⚙️ Требования к окружению

- **ОС**: Windows, Linux или macOS  
- **Python**: версия **3.9 – 3.12**  
- **RAM**: рекомендуется 8+ ГБ для стабильной работы  
- **Диск**: ~5 ГБ свободного места (для модели и кэша)
